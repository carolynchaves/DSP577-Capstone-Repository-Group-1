# Data
This folder contains the dataset used for the project.

## Dataset
Global Climate-Health Impact Tracker (2015–2025)

## Source
https://www.kaggle.com/datasets/sohumgokhale/global-climate-health-impact-tracker-2015-2025/data

## Description
The dataset includes weekly observations across 25 countries and contains variables related to:
- Air quality (PM2.5, AQI)
- Climate conditions (temperature, extreme weather)
- Health outcomes (respiratory disease rates, hospital admissions)
- Socioeconomic indicators

The data is used for exploratory analysis in Python and visualization in Power BI.
